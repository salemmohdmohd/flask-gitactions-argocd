CI/CD Demo with GitHub Actions & ArgoCD

Project Overview

What This Project Does:

  1. Application: Flask API that fetches cat facts and dog images from public APIs
  2. Containerization: Dockerized application for consistent deployment
  3. CI Pipeline: GitHub Actions builds and pushes Docker images to DockerHub on every commit
  4. GitOps CD: ArgoCD monitors the repository and automatically deploys changes to Kubernetes
  5. Kubernetes: Deployed with deployment, service, and ingress manifests

Tech Stack: Flask, Docker, Kubernetes, GitHub Actions, ArgoCD
